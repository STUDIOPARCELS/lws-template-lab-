How to Use the Template Lab

Build a page from your projects, then publish it — one quick loop:

  1. Pick a project (top-left dropdown), then pick a Template next to it — it lays out that project's text.
  2. Add images: click 🖼️ Browse Supabase Images, then click any image to drop it in. Adjust with Image Fit (Fill / Fit) and ⊕ Center on Page. Drag a layer to move it, drag its corners to resize, double-click text to edit.
  3. When the band looks good → click 💾 Save as Page Section (top toolbar, or inside Page Builder).
  4. Repeat for each band (e.g. a hero, a description, a specifications block).
  5. In Page Builder your sections show stacked as the real page. Reorder with ↑ ↓; click ✏️ Edit on a section to fix it, then Save.
  6. Click 👁 Preview to see the whole page full-size.
  7. Click 🚀 Publish, name it, and it downloads the page — send that file to Claude and it goes live at templatelab.lisawoodstudio.com/<name>.

💾 Your work auto-saves in this browser. Click ⬇ Download Full Backup now and then for a safe copy.
